# University Proficiency Exam System (HCI Project)

## 📌 Overview
This project focuses on improving the user experience of the University Proficiency Exam system by designing a centralized and user-friendly platform for students and administrative staff.

The system aims to simplify exam-related processes such as announcements, attendance tracking, and retake registration.

---

## 🚨 Problem
The current system suffers from:
- Fragmented communication across multiple platforms  
- Manual attendance tracking leading to errors  
- Confusing and inefficient retake registration process  
- Increased workload for administrative staff  

---

## 💡 Solution
We designed a web-based system that:
- Centralizes exam announcements  
- Automates attendance tracking  
- Simplifies retake registration  
- Improves usability for both students and staff
